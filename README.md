# sumscan
SumScan - Advanced Security &amp; Reconnaissance Suite

## 🚀 One-Line Automated Installation (Kali Linux)

Open your Kali Linux terminal and run:

```bash
git clone [https://github.com/creationbd5/SumScan.git](https://github.com/creationbd5/SumScan.git) ~/SumScan && cd ~/SumScan && chmod +x install.sh && ./install.sh

This will automatically:
> Clone the repository into ~/SumScan.
> Install all required dependencies (python3-tk, customtkinter, pillow, dnspython).
> Generate a Desktop shortcut with logo.
> Launch SumScan immediately.
